# Objective
外部のPC-98／YM2608／PMD版Bad Apple OGGを測定原典として、現在の製品単声出力と再現可能に比較するオフライン音響テストと成果物を作る。

# Constraints
- 第4区分の変更を保持し、音声エンジンを巻き戻さない。
- 参照OGGを実行時資産またはリポジトリ内の音声資産にしない。
- 時間整列と音量整合を行い、単純な全曲平均や音量差を音質差と誤認しない。
- 自動測定を聴感上またはPMD意味論上の正解証明として扱わない。
- 本番APIや高頻度経路を検査目的で変更しない。
